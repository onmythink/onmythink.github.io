25. As shown in the table, the monthly  rent of an apartment depends on the  number of bedrooms.  If the pattern is  extended, which of these is the likely  cost of a 4­bedroom apartment? Bedrooms 1 2 3 Rent $550 $625 $700


Arithmetic Sequence

Given:

[tex] a_{1} [/tex] = $550
[tex] a_{2} [/tex] = $625
[tex] a_{3} [/tex] = $700

Common Difference can be computed using the formula d = [tex] a_{2} [/tex] - [tex] a_{1} [/tex]

Solution:

d = [tex] a_{2} [/tex] - [tex] a_{1} [/tex] 
d = 625 - 550
d = 75

The fourth term of the sequence can be computed using the formula [tex] a_{n} [/tex] = [tex] a_{1} [/tex] + (n-1) d

Solution:

[tex] a_{4} [/tex] = [tex] a_{1} [/tex] + (4-1) d
[tex] a_{4} [/tex] = 550 + (3) 75
[tex] a_{4} [/tex] = 550 + 225
[tex] a_{4} [/tex] = 775

Thus, $775 will likely cost a 4-bedroom apartment. 
Solution:

By Using ARITHMETIC SEQUENCE:

Let a₁ =550, a₂ = 625, a₃ = 700 and a₄ be the cost of a 4­-bedroom apartment.

So, we find the common difference of the sequence:

d = a₂ - a₁
d = 625-550
d = 75

By using the known values we get the value of the cost of a 4-bedroom apartment.;

a₄ = a₁ + (n-1)d
a₄ = 550 + (4-1)75
a₄ = 550 + (3)75
a₄ = 550 + (225)
a₄ = 775

Therefore, the common difference of the sequence is $75 and the cost of a 4­-bedroom apartment is $775.

29. Kats has rented out 12 units of one- and two-bedroom apartments. The one-bedroom costs 3600 pesos to rent and the two-bedroom costs 4500 to rent out. All units were rented out to a total cost of 49,500 pesos. What is the number of one- and two-bedroom apartments rented out? Answer first the number of one-bedroom apartments, then the number of two-bedroom apartments that were rented out.


One-Bedroom unit: x
Two-bedroom unit: y

Equation (1):

x + y = 12
y = 12-x

Equation (2):

3,600x + 4,500y = 49,500

Solve the number of one-bedroom units by substitution:

3,600x + 4,500y = 49,500
3,600x + 4,500 (12 - x) = 49,500
3,600x -4,500x + 54,000 = 49,500
-900x = 49,500 - 54,000
900x/900 = 4,500/900
x =  5  ⇒  Number of units of One-bedroom

Solve for y, the two-bedroom unit:
y = 12-x
y = 12 - 5
y = 7  ⇒   Number of units of Two-bedroom

Check:

Number of Units:
x + y = 12
5 + 7 = 12
12 = 12 (True)

Rent amount:
3,600x + 4,500y = 49,500
3,600 (5) + 4,500 (7) = 49,500
18,000 + 31,500 = 49,500
49,500 = 49,500   (True)
