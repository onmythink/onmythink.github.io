1. what is the function of virtual device drivers​


Answer:

In certain Microsoft operating systems, a virtual device driver is a program that handles software interrupts from the operating system (rather than hardware interrupts) for each of the computer's main hardware devices, including the hard disk drive controller, keyboard, and serial and parallel ports.

Explanation:

The formula for acceleration is initial velocity divided by time. So the solution is 25 m/s / 3s the answer is 8.333 m/s²

Answer:

Given: Speed (v): 25m/s

Time (t): 3.0 seconds

RTF (Related to Find) : Acceleration

Formula: a = vf-vi / t

Substitution:

a= 25/3

= 8.33333333 or 8.3 or 8 m/s^2
