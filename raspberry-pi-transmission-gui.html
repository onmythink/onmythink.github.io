1. what command can I use to safely shutdown my raspberry pi via terminal ssh?​


ANSWER:

You can use the command [tex]sudo\:shutdown\:-h\:now[/tex] if you want to shutdown it immediately. If you want a time interval upon shutting it down, you can use [tex]sudo\:shutdown\:-time\:"the\:time\:you\:want"[/tex].
