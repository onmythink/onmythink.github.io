19. 10abc + 35ab + 20a is equals?


Answer:

5a(2bc + 7b + 4)

Step-by-step explanation:

10abc + 35ab + 20a

Clearly we can see that the common variable to the algebraic expression is a

Then we 'separate' the variable a from the rest

⇒ a(10bc) + a(35b) + a(20)

⇒ a(10bc + 35b + 20)

We can factor out 5 from the expression 10bc + 35b + 20

⇒ 5(2bc + 7b + 4)

Then we have

⇒ 5a(2bc + 7b + 4)
