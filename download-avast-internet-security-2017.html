11. Matt is downloading an 800 kb file from the internet using his 28.8k modem. how long will it take him to finish downloading?


The time it takes Matt to download the file is about 27.7 seconds.

Detail explanation:

A modem or modulator-demodulator is a device to connect a computer or laptop device to the internet network. Each modem has a different speed that affects the internet speed—the greater the internet speed, the faster the download process.

Modems work with two-way communication in converting digital signals into analog signals to the receiving device. Then then the receiving device will convert the received analog signal into a digital signal again.

There are three main functions of a modem:

1. Modulation, converting digital signals into analog signals

2. Demodulation, converting an analog signal into a digital signal

3. Connecting a computer or laptop device to the internet network

Internet access speed is data transfer speed when using the internet network measured in bits, bits per second.

Kbps = Kilobits per second

Mbps = Megabits per second

Gbps = Gigabits per second

It should be noted that the bit must be written start with a lowercase letter (bps, Kbps, Mbps, Gbps) because capitalization indicates a different meaning, Byte. A byte is a unit measure of the data transfer rate on a computer device. Bytes and bits can be converted to each other. Each byte is equal to 8 bits.

Here's the bit unit conversion:

1 Kbps = 1000 bps

1 Mbps = 1000 Kbps

1 Gbps = 1000 Mbps

The 28.8k modem has a speed of 28,800 bps or 28.8 Kbps.

To find out the length of the download time, we can divide the size of the downloaded file by the internet speed.

From the question, we can know that the file size is 800 kb and the modem speed is 28.8 Kbps.

Then 800 kb : 28.8 Kbps = 27.7 second. This result estimates that the internet speed is stable during the download process.
