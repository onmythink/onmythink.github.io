2. I The Math club rented a sound system for their annual mathematicscamp. They also renied a generator in case of power interruption. Afterthe 3-day comp, the club paid a total amount Php 3,000, threedays for the sound system and two days for the generator. If eachis rented for one day, the club should have paid a totalamount of Pop 1,100. What was the daily rental cost of thegenerator?​


Answer:

The daily rental cost of the generator is 150 pesos.

Step-by-step explanation:

Translate the problem into two equations.

Let's assume that the daily rental cost for the sound system is x and the daily rental cost for the generator is y.

They paid 3000 Php when they used the sound system for 3 days and the generator for 2 days.

[tex]3x + 2y = 3000[/tex]

If they rented each for one day, they would've paid only 1100 Php.

[tex]x + y = 1100[/tex]

We can rearrange the second equation in a way that shows us that y is equal to 1100-x.

[tex]y = 1100 - x[/tex]

Now to be able to solve for the value of x in the first equation, substitute the y value in that equation with the known y value in the second equation.

It will look like this:

[tex]3x + (1100 - x) = 3000[/tex]

Now combine like terms.

[tex]3x - x = 3000 - 1100[/tex]

[tex]2x = 1900[/tex]

Divide both sides by 2.

[tex]x = 950[/tex]

We now know that the daily rental cost for the sound system (or x) is 950 Php.

We can now solve for y using the second equation.

[tex]y = 1100 - (950)[/tex]

Subtract.

[tex]y = 150[/tex]

Now we know that the day rental cost for the generator is 150 Php.
