Answer:

80.25

Step-by-step explanation:

Blue rope = b

Red rope = r

Green rope = g

b = 3r

g = 5b

r + b + g = 508.25

r + b + 5b = 508.25

r + 6b = 508.25

r + 18r = 508.25

19r = 508.25

r = 508.25/19 = 26.75

Since b = 3r,

b = 3(26.75) = 80.25
