18. Name the two elements of a paint bucket tool? ​


Answer:

The Paint Bucket Tool in Photoshop Elements fills adjacent pixels that are similar in color value to pixels you select. It fills these pixels with either the selected foreground color or with a pattern. To use the Paint Bucket Tool in Photoshop Elements, click the “Paint Bucket Tool” in the Toolbox. Then look in the Tool Options Bar.
