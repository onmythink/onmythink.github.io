9. A type of rich media that refers to a video that is downloaded to the user's computer while the user is viewing it.


Answer:

Progressive Download

Explanation:

A type of rich media that refers to a video that is downloaded to the user's computer while the user is viewing it is called "progressive download." Progressive download is a technique for delivering video content over the Internet. The video is broken up into small segments, and each segment is downloaded to the user's computer as it is needed. This allows the user to start watching the video before the entire video has been downloaded. Progressive download is a more efficient way to deliver video content than streaming, because the user does not have to wait for the entire video to download before they can start watching it.
