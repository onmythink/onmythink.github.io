21. These are conditionals that are only executed when the condition-under-check succeeds. Select one: a. If ... Else If ... Else b. If ... Else c. Switch d. If


The answer is If statement.

If Statement is a conditional statement in programming language that would execute only if the condition is true or in this case, condition under check succeeds. Succeeds means that the condition is satisfied or met or true. If statement is only applicable to  single condition.

But, if you would like to use more than one condition, you could make use of If-else statement, which contains a true or false value. One condition must be satisfied, otherwise, it will execute the false value.

If you would like to use more than two conditions, the best statement to use is the nested If-else or the If..Else If..Else, which includes two or more condition to be compared upon.  

Switch case statement is used for conditions that are exact and shorter. And it is also suitable for two or more conditions or case.
