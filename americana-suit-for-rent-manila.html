15. At a Music store, four albums are rented for every nine movies that are rented. If 16 albums were rented, how many movies should the store expect to be rented? (answer must be AGONSA and it should be ratio)​


Answer:

36 movies to be rented

Step-by-step explanation:

We can set up the problem using the following proportion:

(number of albums rented) / (number of movies rented) = (4 albums) / (9 movies)

If we know the number of albums rented, we can use this proportion to find the number of movies rented. In this case, we know that 16 albums were rented, so we can substitute that into the proportion:

(16 albums) / (number of movies rented) = (4 albums) / (9 movies)

To find the number of movies rented, we can cross-multiply and solve for the variable:

(16 albums) * (9 movies) = (4 albums) * (number of movies rented)

36 movies = number of movies rented

So the store should expect 36 movies to be rented.

19. A group of tourists spends $153 to rent snorkels and fins. A total of 12 snorkels and 15 pairs of fins are rented. Renting a snorkel costs three times as much as renting a pair of fins. How much does it cost to rent a snorkel?


Answer:

$9

Step-by-step explanation:

Let's take the values separately.

First we know that the group spent $153 to rent a total of 12 snorkels and 15 pairs of fins.

S = Snorkels

F = Fins

12S + 15F = 153

We know that the snorkels costs 3 times the value of the fins. So lets substitute S for 3F.

12(3F) + 15F = 153

Now that we have similar terms we can then proceed.

36F + 15F = 153

51F = 153

We then need to divide both sides by 51.

[tex]\frac{51F}{51}[/tex] = [tex]\frac{153}{51}[/tex]

F = 153/51

F = 3

Now that we know how much the fins costs we can then solve for how much the snorkels cost by multiplying it by 3.

S = 3F = 3 * 3

S = 9

Therefore we have:

$3 for the fins and $9 for the snorkels.

Answer:

Problem Solving

The cost to rent a snorkel is $9.

Step-by-step explanation:

Given that the group of tourists have spent $153 in renting snorkels and fins and there are 12 snorkels and 15 pairs of fins being rented, allow x to be the amount paid for the rent of the snorkels and y to be the rent for pairs of fins then 12x + 15y = 153. Given also that the rent for a snorkel is three times the rent of a pair of fins then x = 3y then, 12(3y) + 15y = 153. Simplify the equation, 36y + 15y = 153. 51y = 153. Solve for y by dividing both sides of the equation by 51 then, y = 153/51 = 3. Therefore, renting a pair of fins costs $3. To compute for the rent of a snorkel, use the equation x = 3y then, x = 3($3) = $9.
