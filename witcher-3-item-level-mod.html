22. using Euler's theorem, solve the linear congruence 5x congruent 3 (mod 14)?​


Answer:

x ≡ 9 mod (14)

Explanation:

By using Eulers Theorem

a^φ(n) ≡ 1 mod (n), gcd (a,n) = 1

Take note that 5 is a prime number and it is not a factor of 14 so gcd (5,14) = 1

Step 1 : Find φ(n)

Using Euler's Theorem formula

φ(n) = n[1 - (1 / Π_p)]

where p = all sets of prime factors of n

Let n = 14

By prime factorization, we can say that the prime factors of 14 is 2 and 7

φ(n) = n(1 / Πp)

φ(14) = 14[(1 - (1 / 2)] • [(1 - ( 1/7)]

φ(14) = 14(1 /2) • (6 / 7)

φ(14) = 14(3 / 7)

φ(14) = 42 / 7

φ(14) = 6

So therefore we can conclude that

5^φ(14) ≡ 5⁶ ≡ 1 mod (14)

Step 2 : Multiply both sides of the linear congruence and extract the powers

5x ≡ 3 mod (14)

5⁶ • 5x ≡ 5⁶ • 3 mod (14)

x ≡ 5⁶ • 3 mod (14)

We can take 5² to statisfy the modulo as its remainder respect to 3 where,

5² ≡ 3 mod (14)

x ≡ (5²)³ • 3 mod (14)

x ≡ (3)² • 1((3 mod (14)))

x ≡ 9((3 mod (14)) • 1 ((3 mod (14))

x ≡ 9 mod (14)

Hence, solved

24. Prove that ∀n∈N, 10ⁿ + 3 × 4ⁿ + 5 ≡ 0 (mod 9).


Answer:

Note that we can rewrite 10ⁿ + 3 × 4ⁿ + 5 as:

[tex]\rm 10^n + 3 \times 4^n + 5 = (1 + 9)^n + 3 \times (1 + 3)^n + 5[/tex]

Binomials in the form (1 + x)ⁿ can be expanded as:

[tex]\begin{aligned} \rm (1+x)^n & =\rm \binom{n}{n} x^0 + \binom{n}{n-1}x^1 + ... + \binom{n}{2}x^{n-2} + \binom{n}{1}x^{n-1} + x^n \\ &\rm = x^n + \binom n 1 x^{n-1} + \binom n 2 x^{n - 2} + ... + \binom{n}{n-1}x^1 + \binom{n}{n}x^0 \end{aligned}[/tex]

Here, we can expand (1 + 9)ⁿ + 3 × (1 + 3)ⁿ + 5 as:

[tex]\begin{aligned} \rm (1+9)^n+3\times(1+3)^n+5 & =\rm 9^n+\binom{n}{1}9^{n-1}+\binom{n}{2}9^{n-2}+...+\binom{n}{n-1}9^1+\binom{n}{n}9^0\\&\rm \ +3(3^n+\binom{n}{1}3^{n-1}+...+\binom{n}{n-1}3+1)+5\\& =\rm 9^n + \binom{n}{1} 9^{n-1}+... +9n+1\\& \rm \ +3(3^n+\binom{n}{1}3^{n-1}+...+3n+1)+5\\& \rm = \big[9^n+\binom{n}{1}9^{n-1}+...+9n\big]+\big[3^{n+1}+\binom{n}{1}3^n+...+9n\big]+9\\& \rm=9\big[9^{n-1}+\binom{n}{1}9^{n-2}+...+n\big]\\& \rm \ \ +9\big[3^{n-1}+\binom{n}{1}3^{n-2}+n\big]+9 \end{aligned}[/tex]Since (1 + 9)ⁿ + 3 × (1 + 3)ⁿ + 5 is the sum of multiples of 9, 10ⁿ + 3 × 4ⁿ + 5 is divisible by 9.

When a number n is divisible by k, then n ≡ 0 (mod k).

∵ 10ⁿ + 3 × 4ⁿ + 5 is divisible by 9, ∴ 10ⁿ + 3 × 4ⁿ + 5 ≡ 0 (mod 9).
