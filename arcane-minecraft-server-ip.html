1. how to assign an ip address to the server​


Answer:

Open the Control Panel.

Click Network and Internet > Network and Sharing Center > Change Adapter Settings.

Right-click the network adapter.

Click Properties.

Double-click Internet Protocol Version 4 (TCP / IPv4) .

Note: To keep the current IP address as main address, you must first register it as a static address.

In the IP address field, enter the current main IP address.

In the Subnet mask: field, enter 255.255.255.255.

In the Default gateway: field, enter 10.255.255.1.

In the Preferred DNS server: field, enter 10.255.255.1.

Click Advanced.

In the IP Addresses section, click Add.

In the IP address: field, enter the additional IP address.

In the Subnet mask: field, enter 255.255.255.255.

Click Add.

Confirm the outstanding dialogs with OK.

Note: You may get the warning that the gateway is in a different subnet. You can ignore it. In this case, simply confirm with YES that the configuration should be saved.

The change will be applied. This can lead to a brief interruption of the Remote Desktop connection.

Note: If you cannot access the server after the change, you can log in to the server using the KVM console.
