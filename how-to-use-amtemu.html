25. franks hockey team attemped 15 more goals than specncrs team. franks team attemped 23 goals, write and solve an equation that can be used to find how many goals spencers team aattemped.


Answer:

Spencer's team attempted 8 goals.

Step-by-step explanation:

Let's use the variable S to represent the number of goals Spencer's team attempted.

According to the problem,

Frank's team attempted 15 more goals than Spencer's team. So we can set up an equation to represent this relationship:

✦ 23 = S + 15

We added 15 to S because Frank's team attempted 15 more goals than Spencer's team, so if we know the number of goals Spencer's team attempted (represented by S), we can find the number of goals Frank's team attempted by adding 15.

To solve for S, we can isolate the variable by subtracting 15 from both sides of the equation:

ㅤ↠ㅤ 23 - 15 = S + 15 - 15

ㅤ↠ㅤ 8 = S

Therefore, Spencer's team attempted 8 goals.

30. In a championship game sfacs converted 70% of their freethrow attemps..if 21 points were scored on freethrows..how many freethrw attemps did they try in all?


Answer:

30 freethrows

Step-by-step explanation:

What must be found is the total number of attempts in freethrow which is the base.

Given:

percentage = 21rate = 70%

Formula:

[tex]base = \frac{percentage}{rate} [/tex]

Solution:

[tex]b = \frac{p}{r} \\ b = \frac{21}{70\%} \\ b = \frac{21}{0.7} \\ b = 30[/tex]

Therefore, SFACS has 30 freethrows in all.
