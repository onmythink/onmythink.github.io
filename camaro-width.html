3.

[tex] = \frac{128 \: km}{hr} \times \frac{1 \: hr}{3600 \: s} \times \frac{1000 \: m}{1 \: km} [/tex]

4.

[tex] = 0.15μm \times \frac{1000 \: nm}{1 \: μm} [/tex]

= 150 nm

5.

A= l x w

A = (5 x 105 m) x (4 x 104 m)

A = (525 m) x (416 m)

A = 218,400 sqm
