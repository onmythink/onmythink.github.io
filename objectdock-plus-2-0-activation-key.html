17. what is x squared plus ten x plus 25 is equal to 0?


Answer:

-5

Step-by-step explanation:

This quadratic equation can be solved using factoring method. As you can see from

x² + 10x + 25 = 0,

it is a perfect square where in (x+5)²=0.

Factor: (x+5)(x+5)=0

Equate both expressions to zero.

x + 5 = 0, therefore, x = -5

(It has the same factors so the answer will still be -5 for the second equation)

Checking of answer by substitution method:

Substitute x=5 to the given equation x² + 10x + 25 = 0.

(-5)² + 10(-5) + 25 = 0

25 - 50 + 25 = 0

-25 + 25 = 0

0 = 0

It satisfies the equation, therefore, x = 5.

28. 2x squared plus 3x minus 2 equals 0


Answer:

x = 1/2 or x = -2

Step-by-step explanation:

Solve by Factoring

[tex] {2x}^{2} + 3x - 2 = 0[/tex]

Factor the left side of the equation

(2x - 1)(x + 2) = 0

Set all factors equal to 0

2x - 1 = 0 or x + 2 = 0

x = 1/2 or x = -2
