6. what property is this If UV=KL and KL=6,then UV=6​


Answer:

The situation is described as the transitive property of Equality

Step-by-step explanation:

Transitive property states that if one value is equal to the second value, and the second value is equal to the third value, then the first value should be equal to the third value.

Different Properties of Equality Numbers

1. Commutative Property

2. Distributive Property

3. Associative Property

4. Reflexive Property

5. Symmetric Property

6. Transitive Property

In notation form, Transitive property is  "if [tex]a=b[/tex] and [tex]b=c[/tex], then [tex]a=c[/tex]".

So the situation "If [tex]UV=KL[/tex] and [tex]KL=6[/tex],then [tex]UV=6[/tex]" is justified by the Transitive property of Equality.

29. explain eggshell mosaic​


Answer:

A form of art that originated in Yamanashi. To make eggshell mosaics, they use paint called “Suihi”, which is used for traditional Japanese paintings, to color eggshells and then break the eggshells with fingers into different shapes, and attach them on a base using an adhesive.

Answer:
