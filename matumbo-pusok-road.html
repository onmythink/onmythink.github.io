2. ipinambili ni malaya ng tiket papuntang iceland ang naipong pera.ano ang pusok ng pandiwa nito?​


Tanong:

>> Ipinambili ni malaya ng tiket papuntang iceland ang naipong pera.ano ang pusok ng pandiwa nito?

Sagot:

IPINAMBILI

Pokus ang tawag sa relasyong pansematika ng pandiwa sa simuno o paksa ng pangungusap. Nakikita ito sa mga taglay na panlapi kaya nagkakaroon ng iba't ibang pokus ang pandiwa ayon sa kung ano ang kaganapan ng pandiwa sa pusisyong pansimuno ng pangungusap.

Mga Halimbawa ng Pokus pandiwa;

Ipinangtimpla ng kape ang kutsara.

Ipinangluto ng ulam ang kawali.

Ipinangsulat ng sagot ang lapis.

Ipinangsaing ng kanin ang kaldero.

Ipinangdilig ng halaman ang tabo.

Mga halimbawa ng pokus ng pandiwa ang mga nakasalungguhit na salita.
