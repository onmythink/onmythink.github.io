8. A pair of trousers is being sold at P750. A discount of 30% is offered for a 2nd pair of trousers, and 50% for a 3rd pair. If the customer bought 3 pairs of a trousers, how much did the customer pay? ​


Answer:

Php 1,650.00

Step-by-step explanation:

pair of trousers = Php 750.00

2nd pair has 30% discount

3rd pair has 50% discount

Pair 1 = Php 750.00

Pair 2 = Php 750.00 (0.30) = Php 225.00 (discount)

Php 750.00 - Php 225.00 = Php 525.00

Pair 3 = Php 750.00 (0.50) = Php 375.00 (discount)

Php 750.00 - Php 375.00 = Php 375.00

Php 750.00 + Php 525.00 + Php 375.00 = Php 1,650.00

25. Stretchy knitted fabric. It was originally made of wool, but now madeof wool, cotton and synthetic fabrics. ​.


Answer:

Jersey

Explanation:

Jersey. Jersey is a soft stretchy, knit fabric that was originally made from wool. Today, the jersey is also made from cotton, cotton blends, and synthetic fibers.

Since knit fabric forms from a series of loops, it's incredibly stretchy and can stretch both in width and length. This fabric type works well for zipperless, form-fitting clothing items. The texture of the knit fabric is also flexible and unstructured, so it will conform to most shapes and drape or stretch over them.
