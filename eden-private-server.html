24. 4. Acts as an intermediary for requests from clients seeking resources from other servers.a. Proxy Serverc. File Serverb. Print Serverd. Application Server​


Answer:

a. Proxy Server

Explanation:

A Proxy Server is a computer machine that acts as an intermediary for requests from clients seeking resources from other servers. That means, it works by intercepting connections between client and server. The main advantages of a Proxy Server is that it provide increased performance and security.
