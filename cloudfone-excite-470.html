10. 1. Dina has 20 peso bills and 50 peso bills in her wallet. The difference of the 50 peso bills and 20 peso bills is not less than Php470. Let x be the number of 20 peso bills and y be the number of 50 peso bills. Which of the following is the appropriate inequality model to use for this problem? A. 50x - 20y > 470 B. 50y20x> 470 C. 50y - 20x ≥ 470 D. 50x20y ≥ 470​


The appropriate inequality model to use for this problem is C. 50y - 20x ≥ 470

This inequality represents the difference between the number of 50 peso bills (y) and the number of 20 peso bills (x) is not less than 470. It correctly expresses the given information that the difference between the two types of bills is not less than Php470.

Option A is incorrect because it represents the product of 50x and 20y which is not related to the problem

Option B is incorrect because the inequality sign is in the wrong direction, it should be greater than or equal to not greater than

Option D is incorrect because it represents the product of 50x and 20y which is not related to the problem

So the correct inequality model is C. 50y - 20x ≥ 470

26. a piece of wood is 470 centimeters long. how long is it in meter?


470 centimeters is equal to 4.7 meters.Let's draw the metric conversion table

Metric Conversion Table : 

Kilo-
Hecto-
Deca-
(Unit)
Deci-
Centi-
Milli

Now, let us make it horizontal.
Kilo - Hecto - Deca - Unit - Deci - Centi - Milli

Now, lets figure out what we are going to convert.
Yes, it is in Centimeters

So let's find on where is the CENTImeters here.
Kilo - Hecto - Deca - Unit - Deci - Centi - Milli

And what unit are we going to convert centimeter?
Yes, it is in meter.

So let's find on where is the METER here
Kilo - Hecto - Deca - Unit - Deci - Centi - Milli

Note: Always remember that meter is a unit of length, because it doesn't have any prefix/es

So from Centimeter to Units, how many moves? To the left or to the right?
Kilo - Hecto - Deca - Unit - Deci - Centi - Milli
                                   ←         ←

You got it, 2 moves to the left.

Remember: If the unit of what we are going to convert has bigger value than what unit we are going to convert it, the move is always going to the left. If the unit of what we are going to convert has smaller value than what unit we are going to convert it, the move is always going to the right.

So let us continue, we got 2 moves to the left.
What does it means?

We have to move the decimal point of the number twice to the left.
But in the number 470, there is no decimal point as we can see.
But thing is that there is! For every whole number, the decimal point is always placed at the end of the number, or after the ones digit.

So 470 = 470.

So we now got that we have to move the decimal point twice to the left.

470.

= 47.0
Move twice
= 4.70

The answer is 4.70 when rounded, it will be 4.7
