11. a toy store,five tricycles are sold for every nine bicycles sold.if 27bicycles are sold,how many tricycles are expected to be sold by the store​


Answer:

x = 15

Therefore, if 27 bicycles are sold, we can expect the store to sell 15 tricycles.

Step-by-step explanation:

We can solve this problem using ratios.

The ratio of tricycles sold to bicycles sold is 5:9. This means that for every 9 bicycles sold, 5 tricycles are sold.

To find how many tricycles are expected to be sold if 27 bicycles are sold, we need to set up a proportion:

5/9 = x/27

where x is the number of tricycles expected to be sold.

To solve for x, we can cross-multiply:

5 x 27 = 9 x

135 = 9x

x = 15

Therefore, if 27 bicycles are sold, we can expect the store to sell 15 tricycles.

26. The ratio of toy planes to toy cars to balls in the toy store is 2:5:8. If there are 540 toys in all, how many of each kind of toys are there?


Answer:

There are 72 toy planes.

There are 180 toy cars.

There are 288 balls.

Step-by-step explanation:

2:5:8 = 540

Let x represent the toys.

2x + 5x + 8x = 540

Combine like terms.

15x = 540

Divide both sides by 15.

15x/15 = 540/15

x = 36


Evaluate x as 36

2x

2(36)

= 72

There are 72 toy planes.

5x

5(36)

= 180

There are 180 toy cars.

8x

8(36)

= 288

There are 288 balls.


Checking:

2x + 5x + 8x = 540

2(36) + 5(36) + 8(36) = 540

72 + 180 + 288 = 540

540 = 540
