1. How to install OS in windows 7 64 bits in the Computer​


Answer:

There are multiple ways of installing Windows 7 64 bit on a computer

Explanation:

Things you will need:

Windows 7 x64 installer (either CD or bootable USB drive)a 64 bit capable computerpatience

Installation steps:

Turn the computer on.Access the BIOS by pressing f2 on startup.Make sure that the CD drive or USB drive (this depends on which one you would use) is at the top of the boot sequence.Insert the installation disk to the CD drive or plug in your bootable USB drive. Save BIOS settings, exit and restart your computer.After a while, the install window should show up.Click install now.Accept the license terms.Click Custom (Advanced).Select the partition on where you want to install Windows. Note that you need at least 10 GB of free space.Follow the prompts.Type in your product key if prompted. You should be able to find this on the box if you bought a CD installer or on your email if you bought a digital copy.Proceed with the setup.

You should now be able to install Windows 7 x64 on your computer. Please do note that Microsoft will stop supporting Windows 7 on January 14, 2020. I hope this helps!

16. How to install video card device drivers in a windows 7​


Answer:

1. Open the Windows Start menu and go to the Control Panel.

2. Click on System and Security and Windows Update.

3. Check for Updates

Click Check for Updates if you don’t see any results yet — that tells the operating system to check for regular Windows Updates and also looks for new drivers.

4. Look for graphics card drivers that mention manufacturer names related to graphics components, such as Intel Graphics, NVIDIA, or AMD Graphics. You may see these in the main view or under the Optional Updates category. If you see one of these graphics card drivers, click on the Install button and wait for the installation to complete.

25. compare and contrast the 32 bit and 64 bit operating system​


Answer:

A major difference between 32-bit processors and 64-bit processors is the number of calculations per second they can perform, which affects the speed at which they can complete tasks. 64-bit processors can come in dual-core, quad-core, six-core, and eight-core versions for home computing. Multiple cores allow for an increased number of calculations per second that can be performed, which can increase the processing power and help make a computer run faster. Software programs that require many calculations to function smoothly can operate faster and more efficiently on the multi-core 64-bit processors, for the most part.


26. how to upgrade from 32 bit OS to 64 bit OS on windows 10


Answer:

windows 10 can run on both 32-bit and 64-bit processor architectures. If you have a desktop or laptop running the 32-bit version, you can upgrade to the 64-bit version without acquiring a new license. The only caveat is that there is no in-place upgrade path to make the switch, making a clean installation of Windows 10 the only viable option
