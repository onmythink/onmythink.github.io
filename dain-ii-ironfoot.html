Answer:

Type of Software:

• Application Software. ...

• System Software. ..

• Programming Software.

• While application software is designed for end-users, and system software is designed for computers or mobile devices, programming software is for computer programmers and developers who are writing code. ...

Application Software:

Application software is a type of computer program that performs a specific personal, educational, and business function. Each program is designed to assist the user with a particular process, which may be related to

productivity, creativity, and/or communication.

System:

A system is a group of interacting or interrelated elements that act according to a set of rules to form a unified whole. A system, surrounded and influenced by its environment, is described by its boundaries, structure and purpose and expressed in its functioning. Systems are the subjects of study of systems theory.

Utility Software:

Utility software is software designed to help to analyze, configure, optimize or maintain a computer. It is used to support the computer infrastructure - in contrast to application software, which is aimed at directly performing tasks that benefit ordinary users.

Compute Basics:

A computer consists of four major parts: the input, output, CPU (central processing unit), and memory. Input consists of anything you will add into the computer (microphone, keyboard, mouse, scanner), and output is how the computer gives back to you (think screen, speakers, etc.).

23. During which phase does the nucleus begin to break down? prophase II metaphase II anaphase II telophase II


During prophase II, the nucleus begins to break down. This is to pave way for the migration of the the chromosomes to the metaphase plate.

The spindle fiber stretch and prepare to attach themselves to the chromosomes. While in meiosis I there was replication of chromosomes, in prophase II no such thing happens. The broken down nuclear membrane reappears once the cell reaches karyokinesis and cytokinesis.
