27. sandra is 18 years older than paolo. in 13 years, sandra will be twice as old as paolo will be the. how old is sandra now?


First, we should assign variables to the given facts. Let:

 

Sandra’s Age = X + 18

Paolo’s Age = X

 

In 13 years, they will be:

 

(X + 18) + 13 = 2(x + 13)

 

X + 18 + 13 = 2x + 26

X + 31 = 2x + 26

X – 2x = 26 - 31

-x/-1 = -5/-1

X = 5

 

Therefore, Sandra is now:

 

Sandra’s Age = 5 + 18 = 23

 

Sandra is now 23 year’s old.

 

Checking their age:

 

23 (Sandra) – 5 (Paolo) = 18


28. Percy is currently twice as old as Nico. If 8 is subtracted from Nico’s age, and 4 is added to Percy’s age, Percy will be four times as old as Nico. What is Percy’s age now?


Percy’s age : 36, Nico'age : 18

Further Explanation  

One variable linear equation is an equation that has a variable and the exponent number is one.  

Can be stated in the form:  

 [tex]\rm \boxed{\bold{ax=b}}[/tex]

or  

ax + b = c,

where a, b, and c are constants, x is a variable  

Whereas the two-variable linear equation is a linear equation that has 2 variables and the exponent is one  

Can be stated in the form:  

[tex]\rm \boxed{\bold{ax+by=c}}[/tex]  

x, y = variable  

The above problem can be stated as the two-variable linear equation, with:

P = Percy's age

N = Nico's age

Percy is currently twice as old as Nico, expressed in the form of algebraic equations :

[tex]\rm P=2N~(eq~1)[/tex]

If 8 is subtracted from Nico’s age, and 4 is added to Percy’s age, Percy will be four times as old as Nico, expressed in the form of algebraic equations :

[tex]\rm 4(N-8)=4+P~(eq~2)[/tex]

we combine the 2 equations above :

[tex]\rm 4(N-8)=4+2N\\\\4N-32=4+2N\\\\4N-2N==32+4\\\\2N=36\\\\N=18\\\\P=2N\RightarrowP=2.18=\boxed{\bold{36}}[/tex]
