Questions: 1. How many students went to Coron only? 2.How many students went to Tubbataha Reef only? 3. How many joined the El Nido trip only? 4. How many did not go to any of the tourist spots? ​


SET OPERATIONS AND CARDINAL NUMBER

The answers:

11 students went to Coron only. 1 student went to Tubbataha Reef only. 4 students went to El Nido only. 16 students did not go to any of the tourist spots.

FURTHER EXPLANATION:

What are set operations?

The set operations are performed on two or more sets to obtain a combination of elements as per the operation performed on them.

There are four main kinds of set operations which are as follows.

Union of setsIntersection of setsComplement a setDifference between sets/Relative Complement

Union of Set

The union of two sets A and B can be given by:

A ∪ B = {x | x ∈ A or x ∈ B}

This operation on the elements of set A and B can be represented using a Venn diagram with two circles.

Intersection of Set

If two sets A and B are given, which consist of elements common to both A and B. Then the intersection of A and B is the subset of universal set U, which can be given by:

A ∩ B = {x : x ∈ A and x ∈ B}

Complement of Set

The Complement of a Set A is the set of all the elements which are the elements of the Universal set but not the elements of A. It is represented by the symbol [tex]A'[/tex] or [tex]A^c.[/tex]

Difference between sets/Relative Complement

The Relative Complement of set A with respect to set B is the set of all the elements present in B but not in A. It is also called the difference between two sets A and B representing “A-B”. Here we are talking about the relation of complements of sets A and B only not related to universal set.

A - B = {x: x∈ A and x∉ B} = A∩B′B - A = {x: x∈ B and x ∉ A} =B∩A′

Example 1:

If A = {1, 2, 3, 4, 5}, B = {2, 4, 6}, U = {1, 2, 3, 4, 5, 6, 7, 8}.

Here are the following operations on sets and the solutions.

A ∪ B = {1, 2, 3, 4, 5, 6}A ∩ B = {2, 4}A′ = {6, 7, 8}A - B = {1, 3, 5}

Cardinal Number of Sets:

In a finite set, the number of distinct elements is known as the cardinal number. The term n(A) stands for "the number of elements in the set A" and is used to denote it.

Example 2:

A = {letters in the word MATEMATIKA}

A = {M, A, T, E, , I, K}

n(A) = 6

The cardinal number of the set and properties relating to difference, union, and intersection.

1. Union of Disjoint Sets

if A ∩ B = ∅ and if A and B are two finite sets, then

n(A ∪ B) = n(A) + n(B)

2. Union of two sets

If A and B are two finite sets, then

n(A ∪ B) = n(A) + n(B) – n(A ∩ B)

3. Union of three sets

If A, B, and C are three finite sets, then

n(A ∪ B ∪ C) = n(A) + n(B) + n(C) – n(A ∩ B) – n(B ∩ C) – n(A ∩ C) + n(A ∩ B ∩ C)

Now, we can complete the given problem with the previous explanation.

Let:

S = set of 50 students on a tour in Palawan province n(S) = 50A = set of 24 students joined the trip to Coronn(A) = 24 B =  set of 18 students joined the trip to Tubbataha Reef n(B) = 18  C = set of 20 students joined the trip to El Nidon(C) = 20A∩B = set of 12 students made a trip to Coron and Tubbataha Reefn(A ∩ B) = 12B∩C = set of 15 students made a trip to Tubbataha Reef and El Nidon(B∩C) = 15A∩C = set of 11 students made a trip to Coron and El Nidon(A ∩ C) = 11A ∩ B ∩ C = set of 10  students saw that three tourist spotsn(A ∩ B ∩ C) = 10  

So, we can get:

n(A ∪ B ∪ C) = n(A) + n(B) + n(C) – n(A ∩ B) – n(B ∩ C) – n(A ∩ C) + n(A ∩ B ∩ C)n(A ∪ B ∪ C) = 24 + 18 + 20 - 12 - 15 -11 + 10 = 34

For a better explanation, consider the Venn diagram present in Fig.1.

1. How many students went to Coron only?

24 - (12 - 10) - (11 - 10) - 10 = 24 - 2 - 1 - 10 = 11

Therefore, 11 students went to Coron only.

2. How many students went to Tubbataha Reef only?

18 - (12 - 10) - (15 - 10) - 10 = 18 - 2 - 5 - 10 = 1

Therefore, 1 student went to Tubbataha Reef only.

3. How many joined the El Nido trip only?

20 - (15 - 10) - (11 - 10) - 10 = 20 - 5 - 1 - 10 = 4

Therefore, 4 students went to El Nido only.

4. How many did not go to any of the tourist spots?

n(A∪B∪C)' = n(S) - n(A∪B∪C)n(A∪B∪C)' = 50 - 34n(A∪B∪C)' = 16

Therefore, 16 students did not go to any of the tourist spots.
