8. As shown in the table, the monthly  rent of an apartment depends on the  number of bedrooms.  If the pattern is  extended, which of these is the likely  cost of a 4­bedroom apartment? Bedrooms 1 2 3 Rent $550 $625 $700


Arithmetic Sequence

Given:

[tex] a_{1} [/tex] = $550
[tex] a_{2} [/tex] = $625
[tex] a_{3} [/tex] = $700

Common Difference can be computed using the formula d = [tex] a_{2} [/tex] - [tex] a_{1} [/tex]

Solution:

d = [tex] a_{2} [/tex] - [tex] a_{1} [/tex] 
d = 625 - 550
d = 75

The fourth term of the sequence can be computed using the formula [tex] a_{n} [/tex] = [tex] a_{1} [/tex] + (n-1) d

Solution:

[tex] a_{4} [/tex] = [tex] a_{1} [/tex] + (4-1) d
[tex] a_{4} [/tex] = 550 + (3) 75
[tex] a_{4} [/tex] = 550 + 225
[tex] a_{4} [/tex] = 775

Thus, $775 will likely cost a 4-bedroom apartment. 
Solution:

By Using ARITHMETIC SEQUENCE:

Let a₁ =550, a₂ = 625, a₃ = 700 and a₄ be the cost of a 4­-bedroom apartment.

So, we find the common difference of the sequence:

d = a₂ - a₁
d = 625-550
d = 75

By using the known values we get the value of the cost of a 4-bedroom apartment.;

a₄ = a₁ + (n-1)d
a₄ = 550 + (4-1)75
a₄ = 550 + (3)75
a₄ = 550 + (225)
a₄ = 775

Therefore, the common difference of the sequence is $75 and the cost of a 4­-bedroom apartment is $775.
