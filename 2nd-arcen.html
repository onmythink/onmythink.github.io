Answer:

To simplify the expression 2x^2 + 3x^2 - 2x + 4, we can first combine the like terms of the variable x^2 together, and then combine any remaining like terms.

2x^2 + 3x^2 - 2x + 4

= (2+3)x^2 - 2x + 4

= 5x^2 - 2x + 4

Therefore, the simplified form of the expression 2x^2 + 3x^2 - 2x + 4 is 5x^2 - 2x + 4.

1. The measure of two supplementary angles are 1st (3x + 15) ⁰ and 2nd (2x+5) ⁰. x = 32, Find the measure of two angles. *

A. 1st ∠ = 111° and 2nd ∠ = 69°B. 1st ∠ = 100° and 2nd ∠ = 80°C. 1st ∠ = 55° and 2nd ∠ = 74°D. 1st ∠ = 120° and 2nd ∠ = 60°

Since they are supplementary, we can equate these both measurements to 180 and simplify. Before finding the measure of 1st and 2nd angles, simply solve for x first.

Given values:

1st (3x + 15)°2nd (2x + 5)°

Solution:

3x + 15 + 2x + 5 = 1803x + 2x + 15 + 5 = 1805x + 20 = 1805x = 180 - 205x = 160x = 160/5x = 32

The value of x is 32. To determine the value of the measurements, substitute the value of x to 32 in each measurements and simplify.

3x + 15; x = 323(32) + 1596 + 15111

The value of the 1st angle is 111. Let us now solve for the second one.

2x + 5; x = 322(32) + 564 + 569

The value of the 2nd angle is 69°.

Proving that they are supplementary angles:

111 + 69 = 180180 = 180

Answer:

The value of the 1st angle is 111 and the 2nd is 69.Letter A

Since they are complementary, we can simply equate these both equations to 90° and solve for x. Just what I did in the first question, solve for x before determining the values of 1st and 2nd angle measurements.

Given values:

Solution:

2x + 5 + x + 4 = 902x + x + 5 + 4 = 903x + 9 = 903x = 90 - 93x = 81x = 81/3x = 27

The value of x is 27. Let us determine the values of 1st and 2nd angles.

2x + 5; x = 272(27) + 554 + 559

The value of the 1st angle is 59°.

x + 4; x = 2727 + 431

The value of the 2nd angle is 31°.

Proving that they are complementary angles:
