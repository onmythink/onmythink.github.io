Write the coefficients of the polynomial x^2 + 5x + 16 in a row, and place a blank line underneath the coefficients:

Write the root, x - 1, to the left of the coefficients, and draw a vertical line next to it:

1|_________

Bring down the first coefficient, which is 1, and write it below the line:

Multiply the root, x - 1, by the first number in the top row, which is 1, and write the result underneath the next coefficient, which is 5:

| 1 5 16

Add the two numbers in the second row, which are 5 and 4, and write the result in the next row:

| 1 5 16

Multiply the root, x - 1, by the last number in the top row, which is 16, and write the result underneath the previous sum, which is 9:
